Kerangka Riset, Big Data & Analitika Komunikasi Kebijakan

Dasar teoritis dan implementasi komputasi yang diterapkan dalam dashboard ini sebagai bukti integrasi ilmu perkuliahan.

1. Integrasi Big Data Ingestion

Menghubungkan arsitektur data media sosial (API Twitter/X, Instagram, TikTok) dan Web Crawling secara real-time ke dalam satu saluran data (Pipeline Ingestion). Data dibersihkan menggunakan teknik data cleaning, deduplikasi, dan standarisasi format agar terhindar dari noise atau bots spam.

- Inflow: Live Streaming API
- Filtering: Noise Reduction Alg.
- Format: JSON Instagram Stream

2. Pemrosesan Bahasa Alami (NLP)

Menerapkan analisis sentimen berbasis deep learning (menggunakan pengenalan token, eliminasi stopwords, serta evaluasi kemiripan semantik). Mesin secara otomatis mengekstrak entitas penting (Named Entity Recognition) seperti lokasi (Arafah, Mina, Madinah) dan organisasi (Kemenag) untuk mendeteksi krisis lokal secara tepat waktu.

- Tokenizer: Regex Words Parser
- Model: Sentiment Softmax Clsf.
- NER: Entity Matcher Framework

3. Analitika Komunikasi Kebijakan

Menjembatani gap riset komunikasi dengan mengintegrasikan teori Agenda Setting (isu mana yang dianggap media paling penting) dan Situational Crisis Communication Theory (SCCT). Analis humas Kemenag RI dapat segera menyesuaikan respon rilis media berdasarkan tingkat keparahan krisis yang diidentifikasi oleh data.

- Agenda Setting Index
- SCCT Crisis Framework Mode
- Stakeholder Mapping Matrix
